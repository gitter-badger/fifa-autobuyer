## Getting Started
This project was built on node.js v5.1.1 using npm v3.3.12.  I would highly recommend using [nvm](https://github.com/creationix/nvm) to make sure your environment mimics the same.

Install dependencies:

    npm install

Start dev server:

    npm run start-dev
